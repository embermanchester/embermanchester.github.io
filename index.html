Ember JS Manchester

For developers in Manchester and the North West of England who are interested in building modern, fast, ambitious web applications using Ember JS.

Who are we?

Ember JS Manchester is a community group for those with an interest in building web applicaitons with the EmberJS JavaScript framework. We are a friendly group and welcome everyone from beginners to advanced users.

What is Ember?

According to the official website Ember is a

framework for creating ambitious web applications. It is built for productivity, designed with developer ergonomics in mind and has friendly APIs which help you get your job done fast.
