Ember JS Manchester

For developers in Manchester and the North West of England who are interested in building modern, fast, ambitious web applications using Ember JS.

Who are we?

Ember JS Manchester is a North West based community group for those with an interest in building web applications with the EmberJS JavaScript framework. We are a friendly group and welcome everyone from beginners to advanced users. Our members also offer developer support and consultancy.

What is Ember?

Ember is a

framework for creating ambitious web applications. It is built for productivity, designed with developer ergonomics in mind and has friendly APIs which help you get your job done fast.

It is a JavaScript based client side MVC framework and uses all the standards you know and love such as CSS and HTML.

Do you need help?

If you would like some help to develop an Ember based application our members also offer design and consultancy services. Please get in touch via twitter or the old fashioned way using email and we will get back to you ASAP.
